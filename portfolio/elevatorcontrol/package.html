This page lists and describes the java files in your elevatorcontrol package.  Update this document to reflect your implementation and add your java files to the elevatorcontrol folder.

Note:  In order for your portfolio to be complete, the code files in the elevatorcontrol folder must compiled and run if copied into the elevatorcontrol package (overwriting any duplicates) of a clean version of the latest required simulator release.

Controller files

  • CarButtonControl.java - Car Button Controller class
  • DoorControl.java - Door Controller class
  • DriveControl.java - Drie Controller class
  • HallButtonControl.java - Hall Button Controller class
  • ...

Translator files

Links and descriptions to all message translators used by the controllers.
  • CarCallCanPayload.java - CAN payload for mCarCall
  • CarLightCanPayload.java - CAN payload for mCarLight
  • DesiredDwellCanPayload.java - CAN payload for mDesiredDwell
  • DesiredFloorCanPayload.java - CAN payload for mDesiredFloor
  • DoorMotorCanPayload.java - CAN payload for mDoorMotor
  • CanPayload.java - CAN payload for mDrive
  • CanPayload.java - CAN payload for mDrive
  • CanPayload.java - CAN payload for mHallCall
  • CanPayload.java - CAN payload for mHallLight
  • ...
